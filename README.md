# Hydrology-Tool-Python
基于Python的水文流域自动化处理脚本。实现了GIS属性表清洗，面积分类统计与Excel报告自动生成。
# 🌊 流域参数自动提取工具 (Basin-Data-Extractor)

本项目是我在处理水文建模前处理数据时开发的自动化方案。

### 🚀 解决的问题
在水文分析中，GIS 导出的原始属性表经常存在：
1. **命名不规范**：如列名大小写混杂、带空格。
2. **处理重复性高**：数千条子流域数据需要手动在 Excel 中进行尺度分级和统计。

### ✨ 核心功能
- **自动化路径寻址**：脚本可自动搜索匹配不同命名的原始 CSV 文件。
- **数据自动清洗**：自动修正列名大小写及首尾空格。
- **参数化统计**：一键计算总面积（本示例中为 10149 km²）、子流域数量及平均面积。
- **自动分类**：基于面积自动划分流域等级（大型/中型/小型）。

### 🛠️ 技术栈
- **语言**: Python
- **核心库**: Pandas (数据处理)
- **环境**: Cursor / Miniconda
